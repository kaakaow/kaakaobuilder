🍫 KaakaoBuilder – Dokumentaatio

Makein tapa tehdä nettisivu selaimessa

🚀 Ominaisuudet

  • Paikallinen tallennus (localStorage)
  • Moniprojekti-tuki
  • Koodi-editori HTML + CSS
  • Teemavaihtoehdot: Kaakao, Vaalea, Yksinkertainen
  • Drag & Drop -muokkaus
  • Offline-käyttö

📁 Rakenne

/
├── index.html          ← Pääsivu / builder
├── dragndrop.html      ← Visuaalinen editori
├── documentation.html  ← Tämä dokumentaatio
└── static/             ← Teemat, kuvat ym.

💡 Käyttöohje

  1. Avaa index.html selaimessa
  2. Luo uusi projekti
  3. Muokkaa koodia ja tallenna automaattisesti localStorageen
  4. Siirry visuaaliseen muokkaukseen painamalla 🎨-painiketta

⚙️ Tallennusrakenne (localStorage)

{
  "id": "p_8fs91jd",
  "name": "Oma sivu",
  "html": "<!doctype html><html>...</html>",
  "css": "body{...}",
  "created": 1728217382000
}

🎨 Drag & Drop -tila

Kun avaat projektin 🎨 (drag & drop) -painikkeella, builder tallentaa avaimen:

localStorage.setItem('kaakao_current_project', projectId);

dragndrop.html osaa automaattisesti hakea tämän projektin muokattavaksi.

📦 Yhteenveto

KaakaoBuilder on yksinkertainen mutta tehokas web-builderi, joka toimii täysin selaimessa. Se on täydellinen kevyisiin projekteihin, demoihin ja oppimiseen — ilman backend-palvelinta.
